桌面宠物 - 便携版使用说明
========================

感谢您使用桌面宠物（便携版）！

本文件夹包含了运行桌面宠物所需的所有文件，无需安装 Java 环境。

【如何运行】
1. 打开 DesktopPet-Portable 文件夹。
2. 双击 "DesktopPet.exe"（或 "DesktopPet.exe.bat" 备用）。
3. 宠物将会出现在您的桌面上。

【无需安装 Java】
- 文件夹内包含 minimal-jre（精简版 Java 运行环境），无需在电脑上单独安装 Java。
- 如果 minimal-jre 文件夹被删除，程序会尝试使用系统已安装的 Java。

【文件说明】
- DesktopPet.exe         主程序（推荐直接运行）
- DesktopPet.exe.bat     启动脚本（备用）
- AdvancedDesktopPet.jar Java 程序包
- chibi01.ico            程序图标
- minimal-jre/           内置 Java 运行环境
- Image/                 程序图片资源
- music/                 程序音乐资源
- create-shortcut.bat    创建桌面快捷方式
- install.bat            简易安装脚本
- README.txt             英文说明
- README_zh.txt          中文说明（本文件）

【常见问题】
1. 双击后无反应？
   - 请确认所有文件都在同一文件夹内，尤其是 minimal-jre 文件夹。
   - 检查杀毒软件是否误报拦截。
   - 尝试右键“以管理员身份运行”。

2. 出现“找不到 Java”提示？
   - 请确保 minimal-jre 文件夹未被删除。
   - 或自行安装 Java（推荐 https://adoptium.net/ ）。

3. 如何创建桌面快捷方式？
   - 双击运行 create-shortcut.bat。

4. 如何卸载？
   - 直接删除 DesktopPet-Portable 文件夹即可。
