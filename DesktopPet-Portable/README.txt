Desktop Pet - Portable Version
==============================

REQUIREMENTS: Java 8 or later
DOWNLOAD JAVA: https://adoptium.net/

HOW TO RUN:
1. Double-click "run_with_check.bat" (recommended)
2. Or double-click "run.bat" (if Java is installed)
3. Or run: java -jar AdvancedDesktopPet.jar

CONTROLS:
- Left-click and drag to move
- Middle-click for settings
- Right-click for animations
- Double-click to jump
